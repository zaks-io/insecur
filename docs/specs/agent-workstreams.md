# Agent Workstreams

This file is retained only for older links. The durable architecture ownership map moved to
[architecture-groups.md](architecture-groups.md).

Current terminology:

- `AG0` through `AG10` are architecture groups: ownership, seams, references, and handoff contracts.
- `W1` through `W8` are Linear Workstream labels: execution lanes for orchestrating non-overlapping
  work. Their policy lives in [workflow/config.md](../agents/workflow/config.md).
